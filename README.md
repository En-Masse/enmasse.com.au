# [EnMasse.com.au](http://enmasse.com.au)

A Gatsby based [AMP](https://www.ampproject.org) website and blog

## Starting

Make sure you install gatsby globally

`npm i gatsby -g`

Then run a npm install

`npm i`

## Developing

`npm run develop`

## Building

`npm run build`
